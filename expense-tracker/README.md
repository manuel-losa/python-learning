# 💸 Expense Tracker (Python)

A console-based expense tracking application built with Python.

---

## 🚀 Features

* Add expenses with name, amount, and category
* View all expenses with clear formatting
* Delete expenses by index
* Calculate total expenses
* Show totals grouped by category
* Data persistence using file storage

---

## 🧠 How it works

The application:

1. Stores expenses in a list of dictionaries
2. Saves data to a file (`expenses.txt`)
3. Loads data when the program starts
4. Processes and displays results using modular functions

---

## 🛠️ Structure

* `main.py` → user interaction and menu
* `logic.py` → core logic and data handling
* `expenses.txt` → data storage

---

